# CSCE 102 Grader Helper

This is for CSCE 102 Lab Instructors at the University of South Carolina

## Instructions
- In Gradebook in Blackboard, choose the Lab you want to grade
- Select option to "Download all" to download a zip file of all submissions
- Run the script in `main.py` in the same folder as the zip file just downloaded
- The script will display the name of the file you are grading and then open the html file in the browser
- Students with multiple submissions will have to be processed manually for now
