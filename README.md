# CSCE 102 Grader Helper

This is for CSCE 102 Lab Instructors at the University of South Carolina

## Instructions
- In Gradebook in Blackboard, choose the Lab you want to grade
- Select option to "Download all" to download a zip file of all submissions
- Run the script in `main.py` in the same folder as the zip file just downloaded
- The script will display the name of the file(s) you are grading and then open the html file(s) in the browser
- The script will open the student's latest submission if there are multiple submissions

## Dependencies
- Google Chrome (Windows)
  - The path to the executable must be in the PATH variable, so that it can be invoked via
    the command line
- Windows PowerShell (Windows)
- Firefox (Linux)
  - Chrome can easily be configured for Linux as well
